# graphics-editor-c
Problem Statement: Designed and implemented a menu-driven 2D Graphics Editor in C using a 2D character array as the drawing canvas. The canvas initially filled with the character _ (underscore), and graphical objects should be drawn using the character * (asterisk).

Functions are written to draw a circle, rectangle, line, and triangle.
Able to
      add objects to the picture,
      delete objects from the picture, and
       modify objects in the picture.
Used a 2d array of characters to store the picture.
A function was written to display the picture.
